# GandengTangan — Project Overview

## Project Name

GandengTangan

## Project Type

MVP web application for inclusive promotion and marketing of physical products made by people with disabilities.

## Core Idea

GandengTangan is a web platform that helps promote and market physical handmade products created by people with disabilities. The system is intentionally simple for the MVP version:

- public users can browse products without logging in
- only admin can log in and manage the system
- product ordering is handled through WhatsApp admin
- no internal checkout or payment gateway is included

## Main Goal

Build a simple, credible, accessible, and functional web platform that:

1. introduces disability-made products to the public
2. helps support economic empowerment
3. allows people to browse products easily
4. increases trust through transparent product and maker information

## MVP Positioning

This project is an MVP for competition/demo purposes:

- focus on core features only
- keep the implementation simple
- avoid building complex features outside the agreed scope
